# Clocked — Daily Prompts Feature Concept
*Saved 2026-04-25 | Status: CONCEPT ONLY — do not deploy*

## Core Concept
Send users 1 daily workplace prompt via notification.
Goal: fast, honest responses + daily habit building.

Prompt must be:
- Quick (under 10 seconds)
- Relevant to their job/industry
- Slightly real/raw (not corporate tone)

## Prompt Structure
Each prompt includes:
1. Hook (short, casual) — "Be honest…" / "Quick check:" / "No corporate answer:"
2. Core Question — Real workplace situation (pay, stress, managers, customers, workload, etc.)
3. Response Type (rotates daily)
4. Social Hook — "See how others answered" / "Unlock responses after posting"

## Main Response Types

### 1. Emoji Slider (Primary Feature)
Replace number sliders with a 5-point emoji scale:
- 😄 Very Good
- 🙂 Slightly Good
- 😐 Neutral
- 🙁 Slightly Bad
- 😡 Very Bad

Feel: smooth slider, NOT buttons.

Used for:
- "How was your shift today?"
- "How was management today?"
- "How stressful was work?"

### 2. Tap Poll
Fast multiple choice (3–4 options)

Example: "What slowed you down most today?"
- Understaffed
- Management
- Customers
- System issues

### 3. Yes / No
Used for strong engagement

Example: "Did you get a real break today?"

### 4. Optional Short Response
Used sparingly

Example: "What's one thing customers don't understand?"

## Prompt Selection Logic
- DO NOT assign fixed themes to days
- Randomize prompts daily
- Pull from large categorized pool: Pay, Management, Workload, Customers, Fairness, Stress, Culture
- Don't repeat recent prompts
- Mix response types
- Slightly adapt based on user's industry

## Industry Personalization
- **Restaurant:** Tips, rush, kitchen vs front, managers, customers
- **Retail:** Sales goals, returns, staffing, corporate rules
- **Trades (HVAC, electrical, etc.):** Job difficulty, tools, safety, time pressure
- **Office:** Meetings, workload, managers, productivity

## Engagement Mechanics
- Must answer to unlock results
- Show: percent breakdown + short anonymous responses ("hot takes")
- Streak system (daily participation)
- Simple reactions (like/agree)

## Tone Rules
**Must feel:** Real, slightly edgy, relatable, short
**Avoid:** Corporate/HR language, long explanations

## Example Prompts
- "Be honest—how was your shift today?" → Emoji slider
- "Did management actually help today?" → Yes/No
- "What slowed you down most?" → Poll
- "Did today feel worth the pay?" → Emoji slider
- "Are you overworked here?" → Yes/No

## System Behavior
Each day:
1. Select random prompt category
2. Select prompt not recently used
3. Match to user's industry
4. Assign response type (rotate)
5. Send notification at relevant time

Track:
- Response rate
- Skip rate
- Engagement per prompt

Promote high-performing prompts, remove weak ones.
