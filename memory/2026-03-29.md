# Memory — 2026-03-29 (Clocked App Sprint)

## Clocked App — Major Changes Today

### Frontend Deployment Issue (IMPORTANT)
- Railway caches frontend builds aggressively — changing code alone doesn't bust the cache
- To force a fresh build: update `package.json` `_buildTs` field with `Date.now()`
- To force new filename hash: bump version suffix in `vite.config.js` (v3→v4→v5→v6)
- Both techniques needed together for guaranteed fresh deploy
- Current vite config uses `v5` suffix (as of end of day)
