# Memory — 2026-04-29

## Clocked — Daily Prompts Feature (full build session)

### What was built today
Lukas and I built the entire Daily Prompts feature end-to-end. Here's the final state:

#### Voting Rules
- One vote per user per day GLOBALLY — `UNIQUE(prompt_date, user_id)` in `prompt_responses`
- Vote is tied to one occupation (`industry` column stores which one)
- Switching tabs does NOT carry answers over — each tab checks independently
- After submitting → redirect to `/daily-prompts/feed` (replace history, no back button)
- If already voted and navigate to Daily Prompts → auto-redirects to feed

#### Change Answer Flow
- "Change my answer" button on the pinned post card → navigates to `/daily-prompts?change=true&occ=<occupation>`
- Change mode: bypasses auto-redirect, pre-selects existing answer, shows voting buttons
- Button reads "Save Changes" instead of "Submit Answer"
- After saving → back to `/daily-prompts/feed`

#### Feed Screen (`/daily-prompts/feed`)
- Filter bar: "🌐 All Industries" | "👥 Friends" (mutual followers)
- User's voted post pinned first, other occupations below
- Other occupation posts are view-only for voting (no yes/no buttons)
- Each card: category badge (top-left), occupation pill (top-right), hook text, question, result bars with emojis, engagement row

#### Post Cards (`PromptPostCard.jsx`)
- Slider emojis: 😡 Very Bad / 😕 Bad / 😐 Okay / 🙂 Good / 😄 Great
- Like/dislike/comment buttons match EXACTLY to normal post pill style (same inline styles from PostCard.jsx)
- Save + Share + Flag on right side
- White background in light mode, elevated in dark mode
- Full 740px width on desktop (same as polls feed)
- Comment sheet: bottom sheet overlay with anon IDs, post button, enter to send

#### DB Tables Added
- `daily_prompt_reactions` — `(prompt_date, occupation, user_id, type)` for like/dislike/save
- `daily_prompt_comments` — comments per occupation post per day

#### Backend Routes
- `GET /api/daily-prompts/today` — returns `hasVotedToday`, `votedOccupation`, `userResponse`
- `POST /api/daily-prompts/today/respond` — enforces one global vote
- `GET /api/daily-prompts/feed` — all occupation posts, friends filter
- `POST /api/daily-prompts/feed/:date/:occupation/react` — like/dislike/save toggle
- `GET/POST /api/daily-prompts/feed/:date/:occupation/comments` — comment CRUD
